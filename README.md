## Overview
**Olden Era Template Generator** is a convenient and powerful tool designed to quickly generate high-quality project templates for roleplay, creative writing, game development, and storytelling in the "Olden Era" style.

Save time and maintain consistency with professional-grade templates.

## Key Features
- One-click template generation
- Multiple high-quality templates included
- Customizable parameters
- Olden Era / Medieval / Fantasy style focus
- Clean and organized folder structure
- Character sheets, lore documents, and worldbuilding templates
- Regular updates with new templates
- Easy to use graphical interface

## How to Use
1. Download the latest release
2. Extract the archive
3. Run `Olden-Era-Template-Generator.exe`
4. Choose your desired template type
5. Customize settings
6. Generate and export your project
